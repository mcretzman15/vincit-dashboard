# Vincit Group Sales Pipeline Executive Dashboard

Real-time HubSpot sales pipeline dashboard for Vincit Group executive leadership.

## Features

- Pipeline Overview with KPI cards, charts, team filtering, expandable owner drill-downs
- New Deal form with auto-generated deal names

## Tech Stack

- Next.js 14 (App Router), Recharts, Tailwind CSS, HubSpot CRM API

## Environment Variables

Set HUBSPOT_ACCESS_TOKEN in Vercel project settings.
